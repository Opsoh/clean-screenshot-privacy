# clean-screenshot-privacy
Privacy policy for the Clean Screenshot iOS app.

Clean Screenshot does not collect, store, or share any personal data.
The app operates entirely on-device and does not require user accounts, analytics, advertising frameworks, or third-party tracking.
Information Collection

Clean Screenshot does not collect personal information, usage data, or identifiers of any kind.
Photos Access

If granted by the user, Clean Screenshot accesses photos solely to allow the user to select screenshots and save cleaned images. Photo access is used only for this purpose and is never shared or transmitted.
Third-Party Services

Clean Screenshot does not use third-party analytics, advertising, or tracking services.
Data Sharing
No data is collected or shared.
